- 전에 풀었던 코드의 내용을 쉽게 이해할 수 있게 기록합니다. (주석 사용)

### 🔖 문제 선정 방법

- 문제 선정은 동아리원별로 돌아가면서 합니다.

- 동아리원 모두가 어려웠다면 똑같은 수준의 문제를 반복적으로 풀어보며 잘 풀 수 있을 때까지 풉니다.
- 백준 티어 기준 `골드`를 목표로 합니다.
